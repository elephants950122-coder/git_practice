關於我

你好！我是林建宇，目前就讀於靜宜大學資管系。雖然高中時期是社會組背景，但進入大學後，我在大一的課程中逐步克服了對硬體與程式語法的陌生感，打下了相當扎實的程式基礎。比起害怕寫扣，我現在更享受解決問題的過程，並致力於開發出「真正有用」的軟體與應用。

技術專長與專案經驗

  • 網頁前端與後端開發： 熟悉 Python (Flask) 框架與 Firebase 雲端資料庫的整合。曾實際開發過系上教授資訊的網頁搜尋系統，包辦 HTML 模板設計與後端資料串接。
  • Android App 開發： 具備 Kotlin 與 Android Studio 開發經驗，熟悉 MVVM 架構設計，曾參與開發支援社會福利機構的麵包預約應用程式。
  • 物聯網 (IoT) 實作： 能將軟硬體結合，曾使用 Arduino 搭配實體搖桿，從無到有寫出俄羅斯方塊遊戲的控制邏輯。

組織與團隊經驗

  • 資管系聯合迎新 副總召： 負責協助總召協調系上大型活動策劃。在籌備過程中，我培養了跨部門溝通、團隊協作以及專案進度管控的能力，確保活動能順利推進。

休閒嗜好

在坐在電腦前寫程式之餘，我非常重視維持身心的活躍與健康：

  • 團隊運動： 每週固定安排排球與籃球的練習，享受團隊合作與揮灑汗水的樂趣。
  • 徒手健身： 熱衷於自我挑戰，目前專注於引體向上（Pull-ups）的訓練與突破。
